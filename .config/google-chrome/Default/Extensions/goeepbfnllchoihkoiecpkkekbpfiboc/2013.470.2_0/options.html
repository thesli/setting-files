Where do you care about the weather? Location formatting tips:
  • Don't use old style abbreviations like Conn. for Connecticut, use CT
  • If you are searching for an international city, try searching Weather Underground for your country or province to find acceptable cities
  • Don't use provinces for non-US cities (ie: Vancouver, BC)
  • Zipcodes only work in Canada, UK and the US
