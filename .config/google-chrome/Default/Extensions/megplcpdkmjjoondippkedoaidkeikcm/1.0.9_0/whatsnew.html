What's New for DCCT (Double Click Closes Tab)

VersionNew features
1.0.9
  • Fixed occasional unexpected closing of tab when right clicking just once (IF user chose the 'double right click' to close tab option).
1.0.8 Highly requested feature put in!
  • Double right click to close tab is now an option! - I had a change of heart since yesterday and checked & found that Chrome FINALLY did fix the issue I observed before the prevented using double right click to close tab - so I put the feature in in this version. Please enjoy! BTW I explain what that issue was - a silly oversight by Chrome developers before that they now seemed to have fixed it exactly the same way as I said it could be fixed (shift one pixel like Firefox).
1.0.7 Important maintenance release
  • Chrome starts to enforce many code changes to move to manifest v2 - v1 addons will be delisted later. So I put those changes in.
  • No change in main features - sorry but I really don't have time to investigate if Chrome's limited API now supports double right click or double click on tabs - to those people who kept asking the same questions: I didn't put these features in because Chrome didn't support it before. I doubt that it supports it now. I personally use Firefox to browse, not Chrome. I have 5 Firefox addons to take care of, most of them more powerful than this one and two much more elaborate ones. That's why I couldn't do much else in my spare time.
1.0.6 Minor bug fix release
  • Fixed some links do not open correctly when click and ctrl-click behaviors are switched.
1.0.5
  • Provided options to disable closing the last tab of application or current window.
  • Provided options to switch the behaviors of click & ctrl-click links.
  • Provided options for icon to either close or duplicate tab.
  • Disabled closing tab when user double clicked inside textbox, checkbox, select (dropdown menu), and file upload field.
1.0.3 Minor bug fix release
  • Allows DCCT to close this "What's New" page.
  • Fixed a typo-introduced bug.
1.0.2
  • Added this "What's New" page (can be turned off in "Options")
  • Changed BrowserAction icon to PageAction (icon can be hidden in "Options")
  • DCCT works on "view-source://" pages now and other small improvements
1.0.1 Minor bug fix release
  • Fixed icon display for https://chrome.google.com/* pages
1.0.0 Just released, features include:
  • Double click anywhere on page to close a tab (will not close if clicked on link or clicks were to select a word). Triple click can be chosen in "Options" and will close even when clicked on word/links
  • Reopen closed tabs by pressing down "shift" key when double clicking
  • BrowserAction icon to indicate if DCCT works on current page or not (clicking on the icon will close the tab!). This actually applies to all mouse/keyboard-interacting Chrome extensions!
